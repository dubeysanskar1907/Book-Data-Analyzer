# 📚 Book Data Scraper & Analyzer

This project scrapes book data from the website **books.toscrape.com** and performs data analysis and visualization using Python.

## 🚀 Project Overview

The goal of this project is to demonstrate:

- Web scraping using BeautifulSoup
- Data processing using Pandas
- Data visualization using Matplotlib and Seaborn

The scraper collects book information such as:

- Title
- Price
- Rating
- Availability

The dataset is then analyzed and visualized.

---

## 🛠️ Technologies Used

- Python
- requests
- BeautifulSoup
- Pandas
- NumPy
- Matplotlib
- Seaborn
